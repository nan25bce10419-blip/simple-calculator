1. Problem Statement
Many beginners struggle with writing basic Python programs that take user input and perform arithmetic operations.  
This project provides a simple calculator to help understand conditional logic, functions, and menu-driven programs.


 2. Scope of the Project
- Useful for beginners learning Python  
- Can be extended into a GUI-based or scientific calculator  
- Helps in understanding user input, loops, and switch-case (match-case)


 3. Target Users
- Students  
- Python beginners  
- Users who need a basic command-line calculator

  4. High-Level Features
- Interactive user menu  
- Arithmetic operations  
- Error handling using try-except  
- Clean output  
- Easy to customize and extend
